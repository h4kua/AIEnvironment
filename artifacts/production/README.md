# Production Assets

Direktori ini mendokumentasikan artefak kanonis yang dipakai runtime produksi.

- Model aktif tetap berada di `models/`
- Report aktif tetap berada di `artifacts/reports/`
- Konfigurasi baseline realtime berada di `artifacts/configurations/realtime_feature_baselines_jakarta.json`
- Detail pemetaan lengkap tersedia di `artifacts/production/catalog.json`

Tujuan struktur ini adalah memisahkan artefak aktif dari eksperimen/duplikasi historis yang kini dipindahkan ke `artifacts/legacy_archive/`.
