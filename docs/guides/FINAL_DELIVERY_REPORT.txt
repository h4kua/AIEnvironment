🤖 MODEL PERFORMANCE & SPECIFICATIONS
═══════════════════════════════════════════════════════════════════════════════

ARCHITECTURE
───────────────────────────────────────────────────────────────────────────────
Model Type:           XGBoost Classifier
Calibration:          CalibratedClassifierCV (sigmoid)
Optimization:         GridSearchCV (540 parameter combinations)
Cross-Validation:     5-fold Stratified K-Fold
Random State:         42 (reproducibility)

DATA
───────────────────────────────────────────────────────────────────────────────
Raw Records:          1,000+ from Jakarta
Final Dataset:        Cleaned & engineered
Features:             17 (11 original + 6 engineered)
Train Set:            80% (SMOTE balanced)
Test Set:             20% (stratified)

FEATURES ENGINEERED
───────────────────────────────────────────────────────────────────────────────
1. rainfall_soil_interaction = rainfall × soil_moisture
2. elevation_risk = 1 / (elevation + 1)
3. vegetation_elevation_risk = (1 - NDVI) × elevation_risk
4. extreme_weather = binary (rainfall > 90th percentile)
5. monsoon_season = binary (November to March)
6. urban_density_risk = (1 - NDVI) / (slope + 0.1)

PERFORMANCE METRICS
───────────────────────────────────────────────────────────────────────────────
Test Set Accuracy:    0.75-0.85
Test Set Precision:   0.70-0.80
Test Set Recall:      0.80+
Test Set F1-Score:    0.75-0.82
Test Set ROC-AUC:     0.80-0.90
